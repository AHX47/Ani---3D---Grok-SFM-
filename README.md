This repository contains the Source Engine port for the **Grok Ani** character model. Originally designed by **LSCherry**, this model has been optimized for use in games like Garry's Mod and Source Filmmaker.

## 📂 Folder Contents
* **models/x/**: Contains the 3D mesh files (`.mdl`, `.vvd`, `.vtx`).
    * `grok ani.mdl`: The primary base model.
    * `grok ani 2nd wear.mdl`: Alternative outfit version.
* **materials/models/twitter/grok ani/**: High-quality textures and material parameters.
    * `.vtf` files: Albedo, Normal, and Phong exponent maps.
    * `.vmt` files: Configuration for surface reflections and light interaction.

## 🛠️ Installation
1. Download the `models` and `materials` folders.
2. Move them into your game's root directory:
   - **Garry's Mod**: `Steam/steamapps/common/GarrysMod/garrysmod/`
   - **SFM**: `Steam/steamapps/common/SourceFilmmaker/game/usermod/`
3. Restart your game or refresh the model browser.

# Grok Ani 3D Model (Source Engine Version)

هذا المستودع يحتوي على ملفات نموذج الشخصية **Grok Ani** المصمم بواسطة **LSCherry**، والمعدل ليعمل على محرك **Source Engine**.

## 📂 محتويات المجلد
* **models/x/**: يحتوي على ملفات الهيكل الثلاثي الأبعاد (`.mdl`, `.vvd`, `.vtx`).
    * `grok ani.mdl`: النموذج الأساسي.
    * `grok ani 2nd wear.mdl`: نسخة بملابس بديلة.
* **materials/models/twitter/grok ani/**: يحتوي على الخامات والأنسجة.
    * ملفات `.vtf`: الصور الفعلية للأنسجة (البشرة، الملابس، الشعر).
    * ملفات `.vmt`: ملفات الإعدادات التي تحدد كيفية تفاعل الضوء مع النموذج.

## 🛠️ التعليمات
1. انسخ مجلد `models` و `materials` إلى مجلد اللعبة الرئيسي (مثل `garrysmod/garrysmod/`).
2. تأكد من الحفاظ على هيكل المجلدات كما هو لضمان ظهور الأنسجة بشكل صحيح.
3. إذا ظهر النموذج باللونين الأسود والبنفسجي، تأكد من أن مسار ملفات الـ `.vmt` يطابق المسار المذكور داخل ملف الـ `.mdl`.
